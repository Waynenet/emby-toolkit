# 智能追剧

追新剧、补老剧、查缺失。

## 功能点

- 新剧入库自动加入追剧列表。
- 判断缺失季/集并触发订阅请求。
- 维护剧集状态：在追、暂停、完结等。

## 关键操作

- **一件扫描**：一键把存量剧集纳入智能追剧模块管理
- **扫描缺集**：一键扫描有中间缺集的剧集
- **补全旧季**：一键添加旧番到统一订阅等待订阅补全
- **刷新追剧**：一键刷新在追剧集获取最新元数据和状态

## 片头声纹提取

- **触发方式**：支持关闭、入库即扫、仅扫收藏、播放过半。
- **媒体库范围**：不选则不限制；超大库建议排除。
- **抽样窗口**：片头依次尝试前 3、5、10 分钟；片尾依次尝试后 1、2、3 分钟，命中公共声纹即停止。
- **片尾起点**：可选写入片尾起点，客户端是否跳过由 Emby 决定。
- **片头片尾提取**：任务链可定时补扫实时入库漏掉的活跃季；播放过半模式只实时提取，不回扫历史播放记录。

## 依赖

- Emby 连接正常
- MoviePilot 配置完成（用于订阅）
